# Страница списка проектов

**Path:** `/projects`

---

## Сценарии использования
- [UC-02-01](../usecase/02-project-management.md#uc-02-01): Создание проекта
- [UC-02-02](../usecase/02-project-management.md#uc-02-02): Редактирование проекта
- [UC-02-03](../usecase/02-project-management.md#uc-02-03): Удаление проекта

## Функционал
- Список проектов пользователя (карточки или строки таблицы)
- Каждый элемент: название, дата создания, количество задач
- Кнопка «Новый проект»:
  - Открывает форму (название)
  - При достижении лимита — сообщение с предложением сменить тариф
- Редактирование проекта: inline-редактор или модалка
- Удаление проекта: кнопка с подтверждением (предупреждение о безвозвратном удалении)
- Переход по клику на проект → `/projects/:id/board`
- Ссылка «Настройки» или иконка шестерёнки для каждого проекта → `/projects/:id/settings`
