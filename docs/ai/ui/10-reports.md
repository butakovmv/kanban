## Функционал

### Навигация по отчётам (вкладки)
- **Cumulative Flow Diagram** — график накопленных задач по колонкам во времени
- **Lead Time Distribution Chart** — гистограмма времени выполнения задач
- **Диаграмма Гантта** — временная шкала задач, сгруппированных по Use Case
- **Лог событий** — таблица событий доски

### Каждый график/диаграмма
- Полноэкранный режим просмотра
- Выбор периода (день / неделя / месяц / всё время)
- Легенда

### Лог событий
- Таблица: время, пользователь, тип события, описание
- Пагинация по 100 записей
- Кнопка «Загрузить ещё»
- Фильтр по типу события (опционально)
- Ссылка «На доску» → `/projects/:id/board`
