# Страница настроек проекта

**Path:** `/projects/:id/settings`

---

## Сценарии использования
- [UC-02-02](../usecase/02-project-management.md#uc-02-02): Редактирование проекта
- [UC-02-03](../usecase/02-project-management.md#uc-02-03): Удаление проекта

## Функционал
- Форма редактирования названия проекта
- Кнопка «Сохранить»
- Зона опасных действий:
  - Кнопка «Удалить проект» с двойным подтверждением
  - Предупреждение о безвозвратном удалении всех данных
- Ссылки:
  - «На доску» → `/projects/:id/board`
  - «Управление доступом» → `/projects/:id/access`
