# Страница профиля и настроек

**Path:** `/profile`

---

## Сценарии использования
- [UC-01-04](../usecase/01-auth-and-profile.md#uc-01-04): Привязка TOTP-аутентификатора
- [UC-01-07](../usecase/01-auth-and-profile.md#uc-01-07): Просмотр тарифного плана

## Функционал
- Секция «Безопасность»:
  - Статус TOTP (привязан / не привязан)
  - Кнопка «Привязать TOTP» / «Отвязать TOTP»
  - При привязке: подтверждение текущим паролем, отображение QR-кода, поле для ввода первого кода
- Секция «Тариф»:
  - Название текущего тарифа
  - Лимиты: проекты (текущее / максимум), колонки, исполнители
  - Прогресс-бары или числовые счётчики использования лимитов
- Секция «Сессия»:
  - Кнопка «Выйти» → завершение сессии, перенаправление на `/login`
