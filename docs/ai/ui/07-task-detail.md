## Функционал

### Шапка
- Название задачи (редактируемое)
- Кнопка «Назад к доске» → `/projects/:id/board`

### Основная информация
- **Описание** — текстовое поле (редактируемое, markdown-редактор или plain text)
- **Статус** — выпадающий список колонок (изменение статуса перемещает задачу)
- **Приоритет** — выбор: low / medium / high / critical
- **Исполнитель** — выпадающий список (пользователи + боты)
- **Use Case** — текстовое поле или выбор из существующих
- **Метки** — текстовые чипсы (добавление/удаление)

### Даты
- Дата создания (только чтение)
- Дата взятия в работу (только чтение)
- Дата архивирования (только чтение)

### Прикреплённые файлы
- Список загруженных файлов с возможностью скачивания
- Кнопка «Прикрепить файл» (загрузка с устройства)

### Комментарии
- Список комментариев (автор, дата, текст)
- Поле ввода нового комментария
- Возможность редактирования/удаления своего комментария

### История изменений
- Лента изменений задачи (кто, когда, что изменил)
- Автоматически заполняется системой
