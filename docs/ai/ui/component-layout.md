# Компоненты: Навигация и Layout

## AppLayout

**Страницы:** все, кроме login/register/recover  
**Use Case:** все аутентифицированные

**Поведение:** Каркас страницы: хедер (навигация) + основная область + опциональный сайдбар. Управляет глобальной темой (светлая/тёмная). Содержит роутер-вью.

**Состояния:**
- `loading` — проверка сессии
- `authenticated` — отображён layout
- `unauthenticated` — редирект на `/login`

**Данные:**
- _Store → (app):_ theme ('light' | 'dark'), currentUser
- _API:_ GET /api/v1/auth/session

## Navbar

**Страницы:** все аутентифицированные, внутри AppLayout  
**Use Case:** все

**Поведение:** Верхняя панель. Слева — название приложения / лого. Справа — переключатель темы, аватар пользователя, меню (профиль, выйти). Во вью проекта — дополнительно навигация: доска, документы, отчёты, доступ.

**Состояния:**
- `default` — отображён
- `scrolled` — тень при скролле

**Данные:**
- _User →_: клик по пунктам меню, тоггл темы
- _Store → (app):_ setTheme(theme), logout()
- _API:_ POST /api/v1/auth/logout
