# Страница входа

**Path:** `/login`

---

## Сценарии использования
- [UC-01-02](../usecase/01-auth-and-profile.md#uc-01-02): Вход по логину и паролю
- [UC-01-03](../usecase/01-auth-and-profile.md#uc-01-03): Вход с TOTP вместо пароля

## Функционал
- Форма ввода логина
- Переключатель способа входа: «Пароль» / «TOTP»
- Поле ввода пароля (для режима «Пароль»)
- Поле ввода одноразового кода TOTP (для режима «TOTP»)
- Кнопка «Войти»
- Ссылка «Забыли пароль?» → `/recover`
- Ссылка «Регистрация» → `/register`
- Индикация ошибок аутентификации (неверный логин/пароль/код TOTP)
