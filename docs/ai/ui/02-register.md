# Страница регистрации

**Path:** `/register`

---

## Сценарии использования
- [UC-01-01](../usecase/01-auth-and-profile.md#uc-01-01): Регистрация нового пользователя

## Функционал
- Форма с полями «Логин» и «Пароль»
- Валидация полей (длина, непустые)
- Индикация ошибки при занятом логине
- Кнопка «Зарегистрироваться»
- Ссылка «Уже есть аккаунт? Войти» → `/login`
- После успеха — перенаправление на `/login`
