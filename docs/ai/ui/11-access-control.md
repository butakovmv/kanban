## Функционал

### Список групп
- Карточки или строки с названиями групп
- Кнопка «Создать группу» → форма ввода названия

### Детали группы (при выборе)
- Список участников группы:
  - Аватар + имя/логин
  - Кнопка «Удалить» для каждого участника
- Кнопка «Добавить пользователя» → поле ввода логина или email
- Секция «Права доступа» — таблица разрешений:

| Объект | Просмотр | Создание | Редактирование | Удаление |
|---|---|---|---|---|
| Задачи | ☑ | ☑ | ☐ | ☐ |
| Колонки | ☑ | ☐ | ☐ | ☐ |
| Документы | ☑ | ☑ | ☐ | ☐ |

- Чекбоксы для каждого разрешения
- Изменения применяются немедленно
- Ссылка «На доску» → `/projects/:id/board`
