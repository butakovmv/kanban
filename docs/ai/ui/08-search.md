# Страница поиска по задачам проекта

**Path:** `/projects/:id/search`

---

## Сценарии использования
- [UC-05-03](../usecase/05-filtering-and-search.md#uc-05-03): Поиск по задачам
- [UC-05-04](../usecase/05-filtering-and-search.md#uc-05-04): Расширенный поиск с настройками

## Функционал
- Поле поискового запроса (предзаполняется из параметра URL, если перешли с доски)
- Чекбоксы / тогглы настроек поиска:
  - [x] Включить бэклог
  - [x] Включить архив
  - [ ] Искать в комментариях
- Кнопка «Найти»
- Результаты поиска:
  - Список найденных задач с заголовком, статусом, приоритетом
  - Выделение совпавшего текста в названии/описании
  - Клик по результату → `/projects/:id/tasks/:id`
- Пустое состояние: «Ничего не найдено»
- Кнопка «Вернуться на доску» → `/projects/:id/board`
