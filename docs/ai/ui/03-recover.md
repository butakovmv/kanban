# Страница восстановления доступа

**Path:** `/recover`

---

## Сценарии использования
- [UC-01-05](../usecase/01-auth-and-profile.md#uc-01-05): Восстановление доступа по почте

## Функционал
- Форма с полем «Email»
- Кнопка «Отправить ссылку для восстановления»
- Сообщение об успешной отправке
- После перехода по ссылке — форма для ввода нового пароля
- Кнопка «Сохранить новый пароль»
- Ссылка «Вспомнили пароль? Войти» → `/login`
