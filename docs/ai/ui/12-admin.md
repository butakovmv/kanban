# Страница администрирования

**Path:** `/admin`

---

## Сценарии использования
- [UC-10-01](../usecase/10-admin.md#uc-10-01): Изменение тарифного плана пользователя
- [UC-10-02](../usecase/10-admin.md#uc-10-02): Уведомление о превышении лимита тарифа (отображается на других страницах)

## Функционал
- Поиск пользователя по логину или email
- Карточка/таблица найденных пользователей:
  - Логин
  - Текущий тариф
  - Использование лимитов (проекты, колонки, исполнители)
- Выбор нового тарифного плана из выпадающего списка
- Кнопка «Применить» с подтверждением
- После изменения — обновлённая информация о пользователе
- Доступна только системному пользователю (администратору)
