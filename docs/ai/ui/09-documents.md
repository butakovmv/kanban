# Страница документов проекта

**Path:** `/projects/:id/documents`

---

## Сценарии использования
- [UC-06-01](../usecase/06-documents.md#uc-06-01): Просмотр списка документов проекта
- [UC-06-02](../usecase/06-documents.md#uc-06-02): Загрузка нового документа

## Функционал
- Категоризированный список типов документов:
  - Требования заказчика
  - Требования технического архитектора
  - Требования веб-дизайнера
  - User story
  - Архитектура приложения
  - ADR
  - Диаграммы C4 (контекст, контейнеры, компоненты)
  - Use case (текст + sequence-диаграммы)
  - Контракты (предоставляемые)
  - Контракты (используемые)
  - API-тесты
  - UI-тесты
  - Приёмочные тесты
  - Отчёт по уязвимостям, ревью, тестам, покрытию
- Каждый тип — своя секция или вкладка
- Кнопка «Загрузить документ» для каждого типа
- Для существующих документов: preview (если поддерживается) или ссылка на скачивание
- Для документов-диаграмм: встроенный просмотрщик изображений (PNG/SVG)
- Возможность удаления документа
- Ссылка «На доску» → `/projects/:id/board`
