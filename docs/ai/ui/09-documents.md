# Страница документов проекта

**Path:** `/projects/:id/documents`

---

## Сценарии использования
- [UC-06-01](../usecase/06-documents.md#uc-06-01): Просмотр списка документов проекта
- [UC-06-02](../usecase/06-documents.md#uc-06-02): Создание/редактирование документа

## Функционал
- Документы — это markdown-тексты, хранящиеся внутри системы (не файлы)
- Иерархический путь документа (как файлы в папках), например:
  - `docs/requirements/overview`
  - `docs/adr/001-use-postgres`
  - `architecture/c4-context-diagram`
- Страница со списком документов в виде дерева папок и файлов
- Каждый документ отображается с заголовком, путём, датой обновления
- Разворачивание/сворачивание папок (сегментов пути)
- Кнопка «Новый документ» — форма с полями: путь, заголовок, содержимое (markdown)
- Клик по документу — открывает редактор/просмотрщик markdown
- Возможность удаления документа
- Ссылка «На доску» → `/projects/:id/board`
