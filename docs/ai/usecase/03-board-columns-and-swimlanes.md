# Сценарии использования: Колонки и свимлайны

<a id="uc-03-01"></a>
## UC-03-01: Создание колонки
**Актор:** Участник проекта с правом редактирования  
**Цель:** Добавить новую колонку статуса на доску  
**Предусловия:** Пользователь имеет права на редактирование доски, лимит колонок не исчерпан  
**Постусловия:** Создана новая пустая колонка  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система

    Member->>System: Нажатие «Добавить колонку»
    System-->>Member: Поле ввода названия
    Member->>System: Название колонки
    System->>System: Проверка лимита колонок
    alt Лимит не исчерпан
        System->>System: Создание колонки справа от существующих
        System-->>Member: Колонка отображена на доске
    else Лимит исчерпан
        System-->>Member: Ошибка: превышен лимит колонок
    end
```

**Связанный сценарий:** [US-03-01](../userstory/03-board-columns-and-swimlanes.md#us-03-01)

## UC-03-02: Редактирование колонки
**Актор:** Участник проекта с правом редактирования  
**Цель:** Переименовать колонку  
**Предусловия:** Колонка существует  
**Постусловия:** Название колонки обновлено  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система

    Member->>System: Открытие меню колонки
    Member->>System: Выбор «Редактировать»
    System-->>Member: Поле с текущим названием
    Member->>System: Новое название
    System->>System: Обновление названия колонки
    System-->>Member: Название обновлено на доске
```

**Связанный сценарий:** [US-03-02](../userstory/03-board-columns-and-swimlanes.md#us-03-02)

## UC-03-03: Удаление колонки
**Актор:** Участник проекта с правом редактирования  
**Цель:** Удалить колонку  
**Предусловия:** Колонка существует  
**Постусловия:** Колонка удалена, задачи перенесены в бэклог  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система

    Member->>System: Открытие меню колонки
    Member->>System: Выбор «Удалить»
    System-->>Member: Запрос подтверждения
    Member->>System: Подтверждение
    System->>System: Перенос задач в бэклог
    System->>System: Удаление колонки
    System-->>Member: Колонка удалена с доски
```

**Связанный сценарий:** [US-03-03](../userstory/03-board-columns-and-swimlanes.md#us-03-03)

<a id="uc-03-06"></a>
## UC-03-06: Автоматическое появление свимлайна (frontend-only)
**Актор:** Фронтенд (клиент)  
**Цель:** Отобразить новый свимлайн при появлении задачи с новым Use Case  
**Предусловия:** Задача с новым Use Case появилась в колонке (после перемещения из бэклога или создания)  
**Постусловия:** Свимлайн отображён на доске  

```mermaid
sequenceDiagram
    participant Front as Фронтенд
    participant Store as Pinia Store

    Front->>Store: Задача с новым useCase в колонке
    Store->>Store: Группировка задач в колонках по useCase
    alt UseCase не найден среди существующих свимлайнов
        Store->>Store: Создание нового свимлайна для этого useCase
        Front-->>Front: Свимлайн отображён на доске
    else UseCase уже есть
        Front-->>Front: Задача помещена в существующий свимлайн
    end
```

**Связанный сценарий:** [US-03-06](../userstory/03-board-columns-and-swimlanes.md#us-03-06)

<a id="uc-03-07"></a>
## UC-03-07: Автоматическое удаление свимлайна (frontend-only)
**Актор:** Фронтенд (клиент)  
**Цель:** Убрать свимлайн, когда в колонках не осталось задач с его Use Case  
**Предусловия:** Существует свимлайн с задачами  
**Постусловия:** Пустой свимлайн убран с доски  

```mermaid
sequenceDiagram
    participant Front as Фронтенд
    participant Store as Pinia Store

    Front->>Store: Задача покинула колонку (архив/бэклог)
    Store->>Store: Группировка задач в колонках по useCase
    alt Задач с этим useCase в колонках не осталось
        Store->>Store: Удаление пустого свимлайна из группировки
        Front-->>Front: Свимлайн убран с доски
    else Остались задачи с этим useCase
        Front-->>Front: Свимлайн сохраняется
    end
```

**Связанный сценарий:** [US-03-07](../userstory/03-board-columns-and-swimlanes.md#us-03-07)
