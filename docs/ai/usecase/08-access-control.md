# Сценарии использования: Управление доступом

<a id="uc-08-01"></a>
## UC-08-01: Создание группы пользователей
**Актор:** Владелец проекта  
**Цель:** Создать новую группу для разграничения прав доступа  
**Предусловия:** Проект существует  
**Постусловия:** Создана пустая группа  

```mermaid
sequenceDiagram
    actor Owner as Владелец
    participant System as Система

    Owner->>System: Открытие раздела «Управление доступом»
    Owner->>System: Нажатие «Создать группу»
    System-->>Owner: Поле ввода названия
    Owner->>System: Название группы
    System->>System: Создание группы
    System-->>Owner: Группа отображена в списке
```

**Связанный сценарий:** [US-08-01](../userstory/08-access-control.md#us-08-01)

<a id="uc-08-02"></a>
## UC-08-02: Добавление пользователя в группу
**Актор:** Владелец проекта  
**Цель:** Дать пользователю доступ к проекту  
**Предусловия:** Группа существует  
**Постусловия:** Пользователь добавлен в группу  

```mermaid
sequenceDiagram
    actor Owner as Владелец
    participant System as Система

    Owner->>System: Выбор группы
    Owner->>System: Нажатие «Добавить пользователя»
    System-->>Owner: Поле для логина / email
    Owner->>System: Логин или email
    System->>System: Поиск пользователя
    alt Пользователь найден
        System->>System: Добавление в группу
        System-->>Owner: Пользователь добавлен
    else Пользователь не найден
        System-->>Owner: Ошибка: пользователь не найден
    end
```

**Связанный сценарий:** [US-08-02](../userstory/08-access-control.md#us-08-02)

<a id="uc-08-03"></a>
## UC-08-03: Удаление пользователя из группы
**Актор:** Владелец проекта  
**Цель:** Отозвать доступ пользователя к проекту  
**Предусловия:** Пользователь состоит в группе  
**Постусловия:** Пользователь удалён из группы  

```mermaid
sequenceDiagram
    actor Owner as Владелец
    participant System as Система

    Owner->>System: Выбор группы, поиск пользователя в списке
    Owner->>System: Нажатие «Удалить»
    System->>System: Удаление пользователя из группы
    System-->>Owner: Пользователь удалён, доступ отозван
```

**Связанный сценарий:** [US-08-03](../userstory/08-access-control.md#us-08-03)

<a id="uc-08-04"></a>
## UC-08-04: Настройка прав доступа группы
**Актор:** Владелец проекта  
**Цель:** Определить, какие действия участники группы могут выполнять  
**Предусловия:** Группа существует  
**Постусловия:** Права группы обновлены  

```mermaid
sequenceDiagram
    actor Owner as Владелец
    participant System as Система

    Owner->>System: Выбор группы
    Owner->>System: Изменение разрешений (просмотр, создание, редактирование, удаление для задач / колонок / документов)
    System->>System: Применение прав
    System-->>Owner: Права обновлены
```

**Связанный сценарий:** [US-08-04](../userstory/08-access-control.md#us-08-04)
