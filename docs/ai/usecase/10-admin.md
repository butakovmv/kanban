<a id="uc-10-01"></a>
## UC-10-01: Изменение тарифного плана пользователя
**Актор:** Системный пользователь (администратор)  
**Цель:** Изменить тарифный план учётной записи  
**Предусловия:** Целевая учётная запись существует  
**Постусловия:** Тарифный план изменён, новые лимиты активны  

```mermaid
sequenceDiagram
    actor Admin as Администратор
    participant System as Система

    Admin->>System: Поиск пользователя
    System-->>Admin: Данные пользователя, текущий тариф
    Admin->>System: Выбор нового тарифного плана
    Admin->>System: Подтверждение изменения
    System->>System: Применение нового тарифа и лимитов
    System-->>Admin: Успех: тариф изменён
```

**Связанный сценарий:** [US-10-01](../userstory/10-admin.md#us-10-01)

## UC-10-02: Уведомление о превышении лимита тарифа
**Актор:** Система  
**Цель:** Сообщить пользователю о достижении лимита  
**Предусловия:** Пользователь пытается выполнить действие, превышающее лимит тарифа  
**Постусловия:** Действие заблокировано, отображено сообщение  

```mermaid
sequenceDiagram
    actor User as Пользователь
    participant System as Система

    User->>System: Действие (создание проекта / колонки / назначение исполнителя)
    System->>System: Проверка лимита тарифного плана
    alt Лимит исчерпан
        System-->>User: Блокировка, сообщение «Достигнут лимит по <тип> на вашем тарифе»
    else Лимит не исчерпан
        System->>System: Выполнение действия
        System-->>User: Успех
    end
```

**Связанный сценарий:** [US-10-02](../userstory/10-admin.md#us-10-02)
