<a id="uc-04-01"></a>
## UC-04-01: Создание задачи
**Актор:** Участник проекта  
**Цель:** Добавить новую задачу в проект  
**Предусловия:** Пользователь имеет права на создание задач  
**Постусловия:** Задача создана и помещена в бэклог  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система

    Member->>System: Нажатие «Новая задача»
    System-->>Member: Форма (название, Use Case, приоритет, метки, исполнитель)
    Member->>System: Заполненные поля
    System->>System: Создание задачи в бэклоге
    System->>System: Фиксация даты создания
    System-->>Member: Задача отображена в бэклоге
```

**Связанный сценарий:** [US-04-01](../userstory/04-task-operations.md#us-04-01)

<a id="uc-04-06"></a>
## UC-04-06: Редактирование задачи
**Актор:** Участник проекта  
**Цель:** Изменить поля задачи  
**Предусловия:** Задача существует  
**Постусловия:** Поля задачи обновлены, изменения залогированы  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система

    Member->>System: Открытие страницы задачи
    Member->>System: Изменение полей, сохранение
    System->>System: Проверка лимитов тарифа
    alt Лимиты не нарушены
        System->>System: Обновление задачи
        System->>System: Логирование изменений
        System-->>Member: Успех: данные сохранены
    else Нарушение лимита
        System-->>Member: Ошибка: превышен лимит тарифа
    end
```

**Связанный сценарий:** [US-04-06](../userstory/04-task-operations.md#us-04-06)
