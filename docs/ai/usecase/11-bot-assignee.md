## UC-11-01: Назначение бота исполнителем
**Актор:** Участник проекта  
**Цель:** Назначить LLM-агента исполнителем задачи  
**Предусловия:** В проекте зарегистрирован хотя бы один бот, задача существует  
**Постусловия:** Бот назначен исполнителем задачи  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система
    participant Bot as Бот (LLM-агент)

    Member->>System: Открытие задачи
    Member->>System: Выбор бота в поле «Исполнитель»
    System->>System: Назначение бота исполнителем
    System-->>Member: Бот отображён как исполнитель на карточке
    Note over Bot: Бот может быть назначен на другие задачи
```
