# Сценарии использования: Документы проекта

---

<a id="uc-06-01"></a>
## UC-06-01: Просмотр списка документов проекта
**Актор:** Участник проекта  
**Цель:** Увидеть все документы, связанные с проектом  
**Предусловия:** Проект существует, пользователь имеет доступ  
**Постусловия:** Отображён список типов документов  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система

    Member->>System: Открытие вкладки «Документы»
    System-->>Member: Список типов документов (требования, ADR, диаграммы, тесты и т.д.)
    Member->>System: Выбор типа документа
    System-->>Member: Содержимое выбранного документа
```

**Связанный сценарий:** [US-06-01](../userstory/06-documents.md#us-06-01)

## UC-06-02: Загрузка нового документа
**Актор:** Участник проекта с правом редактирования  
**Цель:** Добавить новый документ к проекту  
**Предусловия:** Проект существует, пользователь имеет права на редактирование  
**Постусловия:** Документ сохранён и доступен участникам проекта  

```mermaid
sequenceDiagram
    actor Member as Участник
    participant System as Система

    Member->>System: Переход в раздел «Документы»
    Member->>System: Выбор типа документа
    Member->>System: Загрузка файла / создание через редактор
    System->>System: Сохранение документа
    System-->>Member: Документ доступен участникам проекта
```

**Связанный сценарий:** [US-06-02](../userstory/06-documents.md#us-06-02)
