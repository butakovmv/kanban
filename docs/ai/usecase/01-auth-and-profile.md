# Сценарии использования: Аутентификация и профиль

<a id="uc-01-01"></a>
## UC-01-01: Регистрация нового пользователя
**Актор:** Гость  
**Цель:** Создать учётную запись  
**Предусловия:** Гость не аутентифицирован  
**Постусловия:** Создана учётная запись, назначен тариф по умолчанию  

```mermaid
sequenceDiagram
    actor Guest as Гость
    participant System as Система

    Guest->>System: Запрос формы регистрации
    System-->>Guest: Форма (логин, пароль)
    Guest->>System: Логин + пароль
    System->>System: Проверка уникальности логина
    alt Логин уникален
        System->>System: Создание учётной записи
        System->>System: Назначение тарифа по умолчанию
        System-->>Guest: Успех, перенаправление на вход
    else Логин занят
        System-->>Guest: Ошибка: логин уже используется
    end
```

**Связанный сценарий:** [US-01-01](../userstory/01-auth-and-profile.md#us-01-01)

<a id="uc-01-02"></a>
## UC-01-02: Вход по логину и паролю
**Актор:** Зарегистрированный пользователь  
**Цель:** Аутентифицироваться в системе  
**Предусловия:** Учётная запись существует  
**Постусловия:** Открыта пользовательская сессия  

```mermaid
sequenceDiagram
    actor User as Пользователь
    participant System as Система

    User->>System: Запрос страницы входа
    System-->>User: Форма входа
    User->>System: Логин + пароль
    System->>System: Проверка учётных данных
    alt Данные верны
        System->>System: Открытие сессии
        System-->>User: Успех, главная страница
    else Неверный логин или пароль
        System-->>User: Ошибка аутентификации
    end
```

**Связанный сценарий:** [US-01-02](../userstory/01-auth-and-profile.md#us-01-02)

<a id="uc-01-04"></a>
## UC-01-04: Привязка TOTP-аутентификатора
**Актор:** Зарегистрированный пользователь  
**Цель:** Настроить вход по TOTP без пароля  
**Предусловия:** Пользователь аутентифицирован, TOTP ещё не привязан  
**Постусловия:** TOTP привязан, теперь доступен вход по логину + коду  

```mermaid
sequenceDiagram
    actor User as Пользователь
    participant System as Система

    User->>System: Переход в настройки безопасности
    User->>System: Выбор «Привязать TOTP»
    System-->>User: Запрос текущего пароля
    User->>System: Текущий пароль
    System->>System: Проверка пароля
    alt Пароль верен
        System->>System: Генерация секретного ключа TOTP
        System-->>User: QR-код, запрос первого кода
        User->>System: Код из аутентификатора
        System->>System: Проверка кода
        alt Код верен
            System->>System: Активация TOTP
            System-->>User: Успех: TOTP привязан
        else Неверный код
            System-->>User: Ошибка: неверный код, повторите
        end
    else Неверный пароль
        System-->>User: Ошибка: неверный пароль
    end
```

**Связанный сценарий:** [US-01-04](../userstory/01-auth-and-profile.md#us-01-04)

<a id="uc-01-05"></a>
## UC-01-05: Восстановление доступа по почте
**Актор:** Гость (забывший пароль)  
**Цель:** Восстановить доступ к учётной записи  
**Предусловия:** Учётная запись существует, указана почта  
**Постусловия:** Пароль сброшен  

```mermaid
sequenceDiagram
    actor Guest as Гость
    participant System as Система
    participant Email as Почта

    Guest->>System: Нажатие «Забыли пароль?»
    System-->>Guest: Поле для ввода email
    Guest->>System: Email
    System->>Email: Отправка ссылки для сброса пароля
    Guest->>System: Переход по ссылке, новый пароль
    System->>System: Обновление пароля
    System-->>Guest: Успех: пароль изменён
```

**Связанный сценарий:** [US-01-05](../userstory/01-auth-and-profile.md#us-01-05)
