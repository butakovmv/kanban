# Пользовательские сценарии: Исполнители-боты

## US-11-01: Назначение бота исполнителем
**Роль:** Участник проекта  
**Цель:** Назначить LLM-агента исполнителем задачи  
**Сценарий:**
1. Пользователь открывает задачу.
2. В поле «Исполнитель» выбирает бота из списка.
3. Бот отображается в панели исполнителей и может быть назначен на задачи.

**Связанный сценарий использования:** [UC-11-01](../usecase/11-bot-assignee.md#uc-11-01)
