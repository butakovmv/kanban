# Пользовательские сценарии: Отчёты

<a id="us-07-01"></a>
## US-07-01: Просмотр Cumulative Flow Diagram
**Роль:** Участник проекта  
**Цель:** Проанализировать поток задач по колонкам во времени  
**Сценарий:**
1. Пользователь открывает раздел «Отчёты».
2. Выбирает «Cumulative Flow Diagram».
3. Система отображает график накопленных задач по каждой колонке за период.

**Связанный сценарий использования:** [UC-07-01](../usecase/07-reports.md#uc-07-01)

<a id="us-07-04"></a>
## US-07-04: Просмотр лога событий
**Роль:** Участник проекта  
**Цель:** Просмотреть историю изменений на доске  
**Сценарий:**
1. Пользователь открывает раздел «Отчёты».
2. Выбирает «Лог событий».
3. Система отображает последние 100 событий доски.
4. Пользователь может нажать «Загрузить ещё», чтобы увидеть предыдущие 100 событий.

**Связанный сценарий использования:** [UC-07-04](../usecase/07-reports.md#uc-07-04)
