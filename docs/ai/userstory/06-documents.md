# Пользовательские сценарии: Документы проекта

<a id="us-06-01"></a>
## US-06-01: Просмотр списка документов проекта
**Роль:** Участник проекта  
**Цель:** Увидеть все документы, связанные с проектом  
**Сценарий:**
1. Пользователь открывает вкладку «Документы» проекта.
2. Система отображает список доступных типов документов.
3. Пользователь выбирает тип документа для просмотра.

**Связанный сценарий использования:** [UC-06-01](../usecase/06-documents.md#uc-06-01)

<a id="us-06-02"></a>
## US-06-02: Загрузка нового документа
**Роль:** Участник проекта с правом редактирования  
**Цель:** Добавить новый документ к проекту  
**Сценарий:**
1. Пользователь переходит в раздел «Документы».
2. Выбирает тип документа.
3. Загружает файл или создаёт документ через встроенный редактор.
4. Документ сохраняется и становится доступен другим участникам.

**Связанный сценарий использования:** [UC-06-02](../usecase/06-documents.md#uc-06-02)
