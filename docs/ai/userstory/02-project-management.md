# Пользовательские сценарии: Управление проектами

<a id="us-02-01"></a>
## US-02-01: Создание проекта
**Роль:** Аутентифицированный пользователь  
**Цель:** Создать новый проект с канбан-доской  
**Сценарий:**
1. Пользователь нажимает «Новый проект».
2. Вводит название проекта.
3. Система создаёт проект с доской, содержащей три колонки по умолчанию: To Do, In Progress, Done.
4. Пользователь перенаправляется на страницу проекта.

**Связанный сценарий использования:** [UC-02-01](../usecase/02-project-management.md#uc-02-01)

<a id="us-02-02"></a>
## US-02-02: Редактирование проекта
**Роль:** Владелец проекта  
**Цель:** Изменить название или описание проекта  
**Сценарий:**
1. Владелец открывает настройки проекта.
2. Редактирует название и/или описание.
3. Сохраняет изменения.

**Связанный сценарий использования:** [UC-02-02](../usecase/02-project-management.md#uc-02-02)

<a id="us-02-03"></a>
## US-02-03: Удаление проекта
**Роль:** Владелец проекта  
**Цель:** Удалить проект со всем содержимым  
**Сценарий:**
1. Владелец открывает настройки проекта.
2. Нажимает «Удалить проект».
3. Система запрашивает подтверждение.
4. Владелец подтверждает удаление.
5. Система безвозвратно удаляет проект и все связанные данные.

**Связанный сценарий использования:** [UC-02-03](../usecase/02-project-management.md#uc-02-03)
