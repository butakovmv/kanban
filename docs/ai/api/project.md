# API управления проектами

Базовый путь: `/api/v1/projects`

## Эндпоинты

### `GET /api/v1/projects`
Список проектов текущего пользователя.

- **Доступ:** аутентифицированный пользователь (владелец или участник)
- **Параметры:** `?offset=0&limit=50`
- **Ответ:** `200 OK` — `{ "items": [ { "id", "name", "ownerId", "createdAt" } ], "total": int }`
- **Ошибки:** `401`

### `POST /api/v1/projects`
Создание нового проекта.

- **Доступ:** аутентифицированный пользователь
- **Проверки:** лимит проектов по тарифу
- **Тело:** `{ "name": "string (1-200)", "description?": "string (макс. 2000)" }`
- **Ответ:** `201 Created` — `{ "id": "uuid", "name", "ownerId", "createdAt" }`
- **Ошибки:** `401`, `403` — превышен лимит проектов

### `GET /api/v1/projects/{id}`
Детальная информация о проекте.

- **Доступ:** участник проекта (`task:read`)
- **Ответ:** `200 OK` — `{ "id", "name", "description?", "ownerId", "createdAt", "taskCounts": { "backlog": int, "columns": { colId: int }, "archive": int } }`
- **Ошибки:** `401`, `403`, `404`

### `PUT /api/v1/projects/{id}`
Обновление проекта.

- **Доступ:** владелец проекта
- **Тело:** `{ "name?": "string (1-200)", "description?": "string (макс. 2000)" }`
- **Ответ:** `200 OK`
- **Ошибки:** `401`, `403`, `404`

### `DELETE /api/v1/projects/{id}`
Удаление проекта и всех связанных данных.

- **Доступ:** владелец проекта
- **Ответ:** `204 No Content`
- **Ошибки:** `401`, `403`, `404`
