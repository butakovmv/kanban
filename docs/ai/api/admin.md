# API администрирования

Базовый путь: `/api/v1/admin`

Все эндпоинты требуют прав специального системного пользователя (администратора).

## Эндпоинты

### `GET /api/v1/admin/users`
Поиск пользователей по логину или email.

- **Доступ:** системный администратор
- **Параметры:** `?q=string (мин. 2)&offset=0&limit=20`
- **Ответ:** `200 OK` — `{ "items": [ { "id", "login", "email", "tariff": { "name", "limits" } } ], "total": int }`
- **Ошибки:** `401`, `403`

### `PUT /api/v1/admin/users/{uId}/tariff`
Изменение тарифного плана пользователя.

- **Доступ:** системный администратор
- **Тело:** `{ "tariffName": "string" }`
- **Ответ:** `200 OK` — `{ "userId", "tariff": { "name", "limits" } }`
- **Ошибки:** `401`, `403`, `404`
